# Artifact Schema Reference

This project stores run outputs in `runs/*.json` as schema-versioned artifacts.

## Versioning

- Current schema version: `1`
- Field: `metadata.schemaVersion`
- Type definitions:
  - `src/types/artifact.ts`
  - `src/types/benchmark.ts`
  - `src/types/analysis.ts`

Legacy (pre-v1) artifacts are still supported through `src/artifacts/loader.ts`.

## Run artifact (V1)

```json
{
  "kind": "run",
  "id": "run_...",
  "question": "...",
  "metadata": {
    "schemaVersion": 1,
    "createdAt": "ISO timestamp",
    "model": "gpt-...",
    "fastMode": false,
    "pipelinePreset": "standard | research_deep | fast_research",
    "pipelineVersion": "1.0.0",
    "source": "cli"
  },
  "run": {
    "id": "run_...",
    "createdAt": "ISO timestamp",
    "question": "...",
    "steps": [],
    "finalAnswer": "...",
    "metrics": {}
  }
}
```

### `run.steps[]` core fields

- `id`
- `agentName`
- `role` (`solver | skeptic | synthesizer | research`)
- `request` (schema + prompt metadata)
- `rawAttempts`
- `output`
- `error` (optional)

### `run.metrics` key sections

- `confidence`
- `critique`
- `consensus`
- `quality` (research presets)

## Benchmark artifact (V1)

```json
{
  "kind": "benchmark",
  "id": "benchmark_...",
  "question": "...",
  "metadata": { "...": "same metadata model as run" },
  "payload": {
    "runs": 7,
    "runIds": ["run_a", "run_b"],
    "modeCount": 2,
    "modeSizes": [6, 1],
    "divergenceEntropy": 0.59,
    "summary": {}
  }
}
```

### `payload.summary` includes

- `consensus` stats
- `critiqueMaxSeverity` stats
- full-answer mode metrics
- claim-centroid mode metrics (if available)
- stability pairwise stats (`pairs[]`)

## Analysis index artifact

Generated by:

```bash
pnpm analyze
```

Output:

- `runs/analysis-index.json`

Includes:

- normalized run/benchmark summaries
- filter context describing which CLI filters were applied when index was built
- aggregate issue counts and severity
- confidence drift statistics
- confidence/severity correlations
- outlier run detection (`aggregates.outlierRuns`)

Optional exports:

```bash
pnpm analyze -- --csv --markdown --bundle --chunks
```

- `analysis-runs.csv`
- `analysis-benchmarks.csv`
- `analysis-report.md`
- `analysis-bundle.json` (index + parsed artifacts for sharing)
- `analysis-benchmark-pairs.json` (pairwise similarity chunk)

Optional filters while building index:

- `--question`
- `--model`
- `--preset`
- `--fast-mode`
- `--created-after`
- `--created-before`

## Compatibility notes

- Loader ignores derived analysis JSON files (`analysis-index`, bundle, pairwise chunk).
- Unknown/invalid files are listed under `skipped`.
